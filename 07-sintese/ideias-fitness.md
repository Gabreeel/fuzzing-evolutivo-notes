# Ideias - Função de Fitness

## Objetivos da Fitness Function

### Maximizar Cobertura
- Line coverage
- Branch coverage
- Path diversity

### Encontrar Bugs
- Crashes
- Assertions
- Memory errors
- Comportamentos anômalos

### Performance
- Velocidade de execução
- Uso de memória

## Possíveis Funções de Fitness

### Opção 1: Multi-objetivo
- Combinar cobertura + bugs + performance

### Opção 2: Cobertura Ponderada
- Dar pesos diferentes para diferentes tipos de cobertura

### Opção 3: Novidade
- Recompensar inputs que exploram áreas novas
